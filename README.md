# 📒 Contact Book Website

A simple and responsive Contact Book Website built using **Python (Flask)**, **HTML**, **CSS**, **JavaScript**, and **SQLite**. This project allows users to manage their contacts efficiently with complete CRUD (Create, Read, Update, Delete) operations.

## ✨ Features

- ➕ Add New Contact
- 📋 View All Contacts
- 🔍 Search Contacts
- ✏️ Edit Existing Contacts
- 🗑️ Delete Contacts
- 📊 Display Total Number of Contacts
- 💾 Store Data Using SQLite Database
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/contact-book.git
```

2. Move into the project folder

```bash
cd contact-book
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```
http://127.0.0.1:5000
```
